# Project Title

This app allows you to discover and connect with your favorite influencers in one place! Our app consolidates social media profiles, making it easy to explore and follow influencers across various platforms.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

What things you need to install the software and how to install them.

- Node.js
- npm

### Installing

A step by step series of examples that tell you how to get a development environment running.

```bash
$ git clone https://github.com/sheharyarIshfaq/inspir.club.git
```

```bash
$ cd inspir.club
```

```bash
$ npm install
```

Your are good to go, now run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
